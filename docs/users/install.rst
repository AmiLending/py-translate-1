============
Installation
============

From PyPI
~~~~~~~~~

* To install py-translate, simply:

.. code-block:: bash

    $ pip install py-translate


From Source
~~~~~~~~~~~

* Installation from the Git source repository.

.. code-block:: bash

    $ git clone https://github.com/jjangsangy/py-translate.git

* Use setup.py

.. code-block:: bash

    $ python setup.py install
